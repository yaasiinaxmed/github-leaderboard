# Contributing to GitHub Country Leaderboard

Thank you for your interest in contributing! 🙌

## 🧭 Ways to Contribute

- 📥 Report bugs
- 💡 Suggest features
- ✨ Submit UI improvements
- 🧹 Refactor or clean up code
- 📘 Improve documentation

## 🛠 Development Workflow

1. Fork the project
2. Create your feature branch: `git checkout -b feature/my-feature`
3. Commit your changes: `git commit -m 'Add my feature'`
4. Push to the branch: `git push origin feature/my-feature`
5. Open a Pull Request

We’ll review your PR and get back to you as soon as possible!

## 🎯 Code Style

- Use TypeScript
- Follow Tailwind conventions for utility classes
- Use Prettier + ESLint (if configured)

## 📜 Code of Conduct

This project is governed by a [Code of Conduct](./CODE_OF_CONDUCT.md). By participating, you are expected to uphold this.

Thanks again! 🙏
